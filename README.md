# player-intentions

This Monash undergraduate project examines statistical models to understand tennis player shot selection and intentions. Hidden markov models, with two hidden states were fitted and used in the first project, 2017.

The follow-on summer 2018 project is extend the HMM model to account for different player-opponent matchups.

The project will be co-supervised by Di Cook (@dicook) and Stephanie Kovalchik (@skoval).
